## 1. API-Football (api-football.com / RapidAPI)

| 항목                | 내용                                                                                                                                                          |
| ------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **무료 플랜**       | 100 req/day, 모든 엔드포인트 접근 가능                                                                                                                        |
| **Pro**             | $19/월, 7,500 req/day                                                                                                                                         |
| **Ultra**           | $29/월, 75,000 req/day                                                                                                                                        |
| **Mega**            | $39/월, 150,000 req/day                                                                                                                                       |
| **라이브 데이터**   | O (Livescore 엔드포인트 포함)                                                                                                                                 |
| **xG/xA 고급지표**  | X (기본 제공 안 됨). Statistics 엔드포인트에 기본 스탯(골, 어시스트, 패스, 슈팅 등)만 포함. xG는 별도 유료 애드온(Sportmonks 연동이 아닌 별도 제공 확인 필요) |
| **Rate Limit**      | 플랜별 일일 요청 수 제한 (위 표 참조)                                                                                                                         |
| **데이터 커버리지** | 1,050+ 리그/컵, PL 포함 주요 리그 커버                                                                                                                        |
| **데이터 품질**     | 상위권. 꾸준한 업데이트, 커뮤니티 활성                                                                                                                        |
| **장점**            | 가격 대비 요청 수 우수, RapidAPI 통한 간편 연동, 풍부한 엔드포인트(라인업, H2H, 부상, 이적 등)                                                                |
| **단점**            | xG 등 고급 지표 미포함, 일부 리그 통계 데이터 빈약                                                                                                            |
| **MVP 적합도**      | ★★★★☆ — 기본 스탯 + 라이브 데이터 + H2H에 최적. xG는 별도 소싱 필요                                                                                           |

| **Free w/ Livescores** | EUR 12/월, 12개 대회, 20 req/min                                                     |
| **Standard**           | EUR 49/월, 25개 대회, 60 req/min                                                     |
| **Advanced**           | EUR 99/월, 50개 대회, 100 req/min                                                    |
| **라이브 데이터**      | 유료 플랜(EUR 12+)부터                                                               |
| **xG/xA 고급지표**     | X (Statistics Add-On EUR 15/월 추가 시에도 코너, 점유율, 슈팅, 카드 정도. xG 미포함) |
| **Rate Limit**         | 10~120 req/min (플랜별)                                                              |
| **데이터 커버리지**    | 주로 유럽 리그 중심, 최대 100개 대회                                                 |
| **데이터 품질**        | 중간. 커뮤니티 소싱 데이터로 간헐적 불일치 가능                                      |
| **장점**               | PL 무료 접근, 심플한 RESTful API, 가볍게 시작하기 좋음                               |
| **단점**               | xG 미지원, 무료 플랜은 라이브 스코어 불가, 데이터 깊이 제한적                        |
| **MVP 적합도**         | ★★★☆☆ — 초기 프로토타이핑에 유용하나 고급 지표 부재로 프로덕션 한계                  |

## 3. Sportmonks

| 항목                | 내용                                                                          |
| ------------------- | ----------------------------------------------------------------------------- |
| **무료 플랜**       | 덴마크 수페르리가 + 스코틀랜드 프리미어십만 제공 (PL 미포함)                  |
| **Starter**         | EUR 29/월 (연간 EUR 24/월), 5개 리그 선택                                     |
| **Growth**          | EUR 99/월 (연간 EUR 79/월), 30개 리그                                         |
| **Pro**             | EUR 249/월 (연간 EUR 199/월), 120개 리그                                      |
| **xG Add-on**       | EUR 24/월 추가 (xG, xGot, xPTS, npxG 등 다양한 지표)                          |
| **라이브 데이터**   | O (유료 플랜)                                                                 |
| **xG/xA 고급지표**  | O (유료 애드온). Basic(경기 후 12시간), Standard(경기 직후), Advanced(실시간) |
| **Rate Limit**      | 2,000~5,000 req/entity/hour (플랜별)                                          |
| **데이터 커버리지** | 2,300+ 리그, 업계 최고 수준                                                   |
| **데이터 품질**     | 상위권. 전담 스카우트팀의 데이터 검증                                         |
| **장점**            | xG 데이터 전문, 풍부한 엔드포인트, 유연한 리그 선택, 잘 정리된 문서           |
| **단점**            | 무료 플랜에 PL 없음, xG 애드온 별도 비용, 전체적으로 가격 높음                |
| **MVP 적합도**      | ★★★★☆ — xG 포함 시 가장 포괄적이나, PL + xG 최소 EUR 53/월(Starter + xG)      |

| **MVP 적합도**     | ★☆☆☆☆ — MVP 단계에서는 완전히 비현실적. 시리즈 A+ 이후 검토 |

## 5. FPL (Fantasy Premier League) API

| 항목               | 내용                                                                                            |
| ------------------ | ----------------------------------------------------------------------------------------------- |
| **가격**           | 완전 무료                                                                                       |
| **라이브 데이터**  | 부분적 (게임위크 진행 중 데이터 업데이트)                                                       |
| **xG/xA 고급지표** | O (expected_goals, expected_assists, expected_goal_involvements 필드 제공)                      |
| **Rate Limit**     | 비공식 — 명시된 제한 없으나 과도한 요청 시 차단 가능                                            |
| **데이터 범위**    | PL 전용 (시즌 중 모든 선수, 팀, 경기)                                                           |
| **CORS 제한**      | 프론트엔드 직접 호출 불가 — 백엔드 프록시 필수                                                  |
| **장점**           | 완전 무료, PL 모든 선수 xG/xA 포함, ICT Index, 선수 가치/소유율 등 유니크 데이터                |
| **단점**           | 비공식 API(언제든 변경 가능), CORS 차단, PL만 지원, 문서 없음(커뮤니티 문서만), 크립틱한 필드명 |
| **MVP 적합도**     | ★★★★★ — PL 전용 프로젝트에 완벽. 무료로 xG/xA 포함, 백엔드 프록시만 구현하면 됨                 |

## 추천 데이터 소싱 전략 (MVP)

### 1차 추천: API-Football (메인) + FPL API (xG 보충)

**총 비용: $0~29/월**

| 역할             | API                          | 용도                                                                       |
| ---------------- | ---------------------------- | -------------------------------------------------------------------------- |
| **메인 데이터**  | API-Football (Ultra, $29/월) | 라이브 스코어, 경기 일정, H2H, 선수 기본 스탯, 팀 순위, 라인업, 이적, 부상 |
| **xG/xA 데이터** | FPL API (무료)               | 선수별 xG, xA, xGI, ICT Index, 소유율 등 고급 지표                         |
| **xG 보충**      | Understat 스크래핑 (무료)    | 팀별 xG, 슈팅 맵, 시즌 xG 트렌드 (캐시 레이어로 부하 최소화)               |

**구현 아키텍처:**

```
Next.js API Routes (프록시)
├── /api/football/* → API-Football 프록시
├── /api/fpl/*      → FPL API 프록시 (CORS 우회)
└── /api/xg/*       → Understat 데이터 캐시 (Supabase에 주기적 저장)
```

**장점:**

- xG를 포함한 모든 요구사항 충족
- 월 $29 이하로 운영 가능
- FPL API로 xG/xA 무료 확보
- API-Football의 안정적인 라이브 데이터

**단점:**

- 두 API의 선수 ID 매핑 필요
- FPL API 비공식이므로 구조 변경 위험
- Understat 스크래핑 의존

### 2차 추천: Sportmonks (올인원)

**총 비용: EUR 53+/월 (Starter EUR 29 + xG Add-on EUR 24)**

| 역할         | 설명                                        |
| ------------ | ------------------------------------------- |
| **단일 API** | 라이브, 스탯, xG, 일정, H2H 모두 하나의 API |

**장점:**

- 단일 소스로 모든 데이터 해결
- xG 데이터 품질 우수
- ID 매핑 불필요
- 잘 정리된 문서

**단점:**

- 무료 플랜에 PL 없음
- 월 비용 높음 (EUR 53+)
- 연간 계약 시 EUR 48/월로 절감 가능

### 3차 추천: Football-Data.org (프로토타입) → API-Football (프로덕션)

초기 MVP 검증 단계에서 Football-Data.org 무료 플랜으로 빠르게 프로토타입 후, 검증되면 API-Football로 마이그레이션.

## 최종 권장

**pitch-ac MVP에는 1차 추천(API-Football + FPL API) 조합을 추천합니다.**

이유:

1. PL 전용 프로젝트이므로 FPL API의 xG/xA 데이터가 완벽히 매칭됨
2. API-Football이 라이브 데이터 + H2H + 경기 일정을 안정적으로 제공
3. 월 $29 이하로 모든 요구사항 충족 가능
4. Supabase에 FPL/Understat 데이터를 주기적으로 캐싱하면 rate limit 걱정 최소화
5. 선수 ID 매핑 테이블을 Supabase에 구축하면 두 API 간 연동 용이

추후 규모 확장 시 Sportmonks 전환 또는 Opta 검토 가능.
