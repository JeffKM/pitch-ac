# MATCHDAY 자동 동기화 — 킥오프 오프셋 방식

경기 결과가 수동 동기화 없이는 반영되지 않던 문제를 해결하기 위해, DB에 저장된 킥오프 시간 + 2.5시간 오프셋으로 자동 동기화 트리거. football-data.org 무료 플랜(10요청/분) 제약 내에서 동작.

## Considered Options

- **주기적 Cron (하루 2~4회)** — 경기 없는 날에도 불필요한 호출, 경기 직후 반영 불가
- **On-demand (페이지 접속 시 체크)** — 유저에게 수동 행위 요구, UX 저하
- **킥오프 + 오프셋** (채택) — API 호출 최소화, 리그전은 연장 없이 ~2시간 내 종료되므로 정확

## Consequences

- fixtures 테이블의 킥오프 시간을 기반으로 Cron 스케줄을 동적 계산하는 로직 필요
- 경기 일정 자체의 동기화(킥오프 시간 갱신)는 별도 스케줄 필요 (하루 1회 등)
