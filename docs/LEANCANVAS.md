# pitch-ac 린 캔버스 (Lean Canvas)

> 작성일: 2026-03-21
> 기반 문서: [PRD.md](./PRD.md)

---

## 1. 문제 (Problem)

**PL 팬이 겪는 핵심 문제 3가지:**

1. **데이터 파편화** — 선수 스탯, 경기 결과, 라이브 스코어, xG 등을 보려면 FBref, Understat, FotMob, Sofascore 등 여러 사이트를 돌아다녀야 한다.
2. **숫자에 맥락 부재** — "살라 시즌 15골"이라는 숫자만으로는 이것이 좋은 건지, 리그 내 어느 수준인지, 전 시즌 대비 향상된 건지 즉시 판단할 수 없다.
3. **비교·공유의 번거로움** — 두 선수를 비교하려면 수동으로 데이터를 모아야 하고, SNS에 공유할 만한 시각적 결과물을 만들기 어렵다.

**기존 대안:**

| 대안                   | 한계                                                               |
| ---------------------- | ------------------------------------------------------------------ |
| FBref / Understat      | 데이터는 풍부하나 맥락(순위·백분위·전년 비교) 없음, 모바일 UX 열악 |
| FotMob / Sofascore     | 라이브 스코어 중심, 심층 스탯 비교·맥락 해석 부족                  |
| 수동 스프레드시트 비교 | 시간 소모, 공유 불가, 시각화 부재                                  |

---

## 2. 고객 세그먼트 (Customer Segments)

**타깃:** 글로벌 PL(프리미어리그) 팬

**얼리어답터 (초기 핵심 사용자):**

- **데이터 기반 열성 팬** — SNS(X/Reddit)에서 선수 비교 논쟁에 자주 참여하며, xG·xA 같은 고급 지표를 이해하고 활용하는 팬. 비교 근거 데이터와 공유 가능한 시각 자료에 대한 수요가 높음.

**확장 타깃:**

- **캐주얼 팬** — 매치데이 전후 빠른 프리뷰/리뷰가 필요하지만 여러 사이트를 돌아다닐 시간이 없는 팬
- **PL 입문 팬** — xG 등 전문 용어의 의미를 쉽게 이해하고 싶은 신규 팬

---

## 3. 가치 제안 (Unique Value Proposition)

> **"모든 숫자에 맥락을 더한, PL 팬을 위한 원스톱 데이터 플랫폼"**

PL 선수·경기 데이터를 한 곳에서 — 모든 수치에 리그 순위, 포지션 백분위, 전년 비교를 자동으로 붙여 **숫자의 의미를 즉시 해석**할 수 있게 한다.

**하이레벨 컨셉:** "FBref의 데이터 깊이 + FotMob의 모바일 UX + 자동 맥락 해석"

---

## 4. 해결책 (Solution)

문제 3가지에 대응하는 핵심 솔루션:

| 문제                   | 해결책                               | 핵심 기능                                                                                                                                      |
| ---------------------- | ------------------------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------- |
| **데이터 파편화**      | 매치데이 대시보드 + 선수 프로필 통합 | 게임위크 경기 목록(F001), 라이브 스코어 폴링(F002), 프리/라이브/포스트매치 상세(F003~F005)                                                     |
| **숫자에 맥락 부재**   | 맥락 스탯 시스템                     | 선수 프로필 맥락 스탯(F006) — 모든 수치에 리그 순위·백분위·전년 비교 자동 표시, 레이더 차트(F007), 폼 스파크라인(F008), 용어 설명 팝오버(F012) |
| **비교·공유 번거로움** | 선수 비교 배틀카드 + 이미지 공유     | 2인 비교 배틀카드(F009), OG 이미지 생성 및 SNS 공유(F010), Fuzzy 검색 자동완성(F011)                                                           |

---

## 5. 경쟁우위 (Unfair Advantage)

1. **맥락 자동 해석 엔진** — 단순 수치 나열이 아닌, 리그 순위·포지션 백분위·전년 비교를 모든 데이터 포인트에 자동 결합하는 시스템. 기존 서비스가 제공하지 않는 고유한 데이터 표현 방식.
2. **바이럴 내장 구조** — 비교 배틀카드 → OG 이미지 자동 생성 → SNS 공유의 원클릭 파이프라인. 사용자가 콘텐츠를 직접 만들고 퍼뜨리는 구조.
3. **듀얼 API 데이터 소스** — SportMonks(메인) + API-Football(보충) 조합으로 xG, H2H, 부상 정보 등 단일 API로는 커버 불가능한 데이터 폭과 깊이 확보.

## 6. 채널 (Channels)

| 단계       | 채널                                               | 설명                                                          |
| ---------- | -------------------------------------------------- | ------------------------------------------------------------- |
| **인지**   | X(Twitter) / Reddit (r/PremierLeague, r/FantasyPL) | 배틀카드 이미지가 바이럴로 확산 → 워터마크를 통한 브랜드 노출 |
| **인지**   | SEO / 오가닉 검색                                  | "Salah vs Haaland stats", "PL xG rankings" 등 검색 유입       |
| **평가**   | 비교 URL 공유                                      | `pitch-ac.com/compare?p1=X&p2=Y` 형태의 공유 링크로 자연 유입 |
| **활성화** | 매치데이 대시보드                                  | 매 경기일 자동 리디렉션으로 재방문 유도                       |
| **리텐션** | 라이브 스코어 폴링                                 | 경기 중 실시간 데이터 → 세컨드 스크린 습관 형성               |

## 7. 주요 지표 (Key Metrics)

**OMTM (가장 중요한 단일 지표):** 주간 활성 사용자 (WAU)

| 카테고리   | 지표                    | 설명                               |
| ---------- | ----------------------- | ---------------------------------- |
| **획득**   | 신규 방문자 수          | 오가닉 + 바이럴 유입               |
| **활성화** | 경기 상세 페이지 조회율 | 대시보드 → 상세 전환율             |
| **리텐션** | 매치데이 재방문율       | 게임위크별 재방문 비율             |
| **바이럴** | 배틀카드 이미지 공유 수 | Share as Image 클릭 및 다운로드 수 |
| **참여**   | 선수 비교 생성 수       | 비교 페이지 이용 횟수              |

- **무료 서비스** — 초기 사용자 확보와 제품-시장 적합성 검증에 집중

**향후 수익 모델 (MVP 이후):**

| 모델                | 설명                                                            |
| ------------------- | --------------------------------------------------------------- |
| **프리미엄 구독**   | 고급 분석(시즌 트렌드 차트, 3인 비교, AI 예측), 광고 제거       |
| **디스플레이 광고** | 매치데이 대시보드·선수 프로필 페이지 내 광고 (높은 트래픽 활용) |

> MVP에서는 수익보다 WAU 성장과 배틀카드 바이럴 검증에 집중한다.
