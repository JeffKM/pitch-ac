# pitch-ac MVP PRD

## 핵심 정보

**목적**: PL 팬이 선수/경기 데이터를 파편화된 여러 사이트 없이 한 곳에서, 모든 숫자에 맥락(순위·백분위·전년 비교)을 더해 즉시 해석할 수 있게 한다.
**사용자**: 글로벌 PL 팬 — 경기 전후 빠른 프리뷰가 필요한 캐주얼 팬부터 선수 비교 근거와 라이브 스탯을 추적하는 열성 팬까지.

---

## 사용자 여정

```
1. 매치데이 대시보드 (진입점)
   ↓ 앱 첫 접근 시 자동 리디렉션 (현재 게임위크)

2. 게임위크 선택
   ↓ 이전/다음 게임위크 화살표 클릭

   [경기 상태: 예정] → 경기 상세 페이지 (프리매치 탭)
   [경기 상태: 라이브] → 경기 상세 페이지 (라이브 탭, 60초 폴링)
   [경기 상태: 종료]  → 경기 상세 페이지 (포스트매치 탭)
   ↓

3. 경기 상세 페이지
   ↓ 선수 이름 클릭 (라인업/이벤트)

4. 선수 프로필 페이지
   ↓ "Compare" 버튼 클릭 (상대 선수 선택)

5. 선수 검색 페이지
   ↓ 검색 결과에서 선수 선택 → 선수 프로필 페이지 이동
      또는 비교 대상으로 추가 → 선수 비교 페이지 이동

6. 선수 비교 페이지
   ↓ "Share as Image" 클릭

7. 이미지 생성 → 공유 (SNS) / 다운로드
```

## 기술 스택

### 프론트엔드 프레임워크

- **Next.js 16+** (App Router) — React 풀스택 프레임워크, Server Components 우선
- **React 19** — UI 라이브러리 (동시성 기능, use() hook)
- **TypeScript 5.6+** — strict 모드, 타입 안전성 보장

### 스타일링 & UI

- **Tailwind CSS v4** (CSS-first, `@theme inline`, 설정 파일 없음) — globals.css에서 OKLCH 색상 정의
- **shadcn/ui** (new-york 스타일) — 고품질 React 컴포넌트 (Popover, Tabs, Command, Dialog, Progress 등)
- **Lucide React** — 아이콘 (개별 임포트)
- **tw-animate-css** — 애니메이션 유틸리티

### 차트 & 시각화

- **Recharts** — 레이더 차트, 스파크라인 (Client Component)
- **@vercel/og** — OG 이미지 동적 생성 (비교 배틀카드)

### 상태관리 & 데이터 패칭

- **TanStack Query v5** — 서버 상태, API 캐싱, 라이브 폴링 (refetchInterval)
- **Zustand v5** — 클라이언트 상태 (비교 선수 목록, UI 상태)

### 폼 & 검증

- **React Hook Form v7** — 폼 상태 관리
- **Zod v4** — 스키마 검증

### 백엔드 & 데이터베이스

- **Supabase** — PostgreSQL DB, 인증 (이메일 + Google OAuth 구현 완료), `@supabase/ssr` 기반
- **SportMonks API** (메인, European Basic €39/mo) — League ID: 8, 경기/라이브스코어, 선수 스탯, xG, 라인업, 부상, 순위, includes 시스템으로 중첩 데이터 조회
- **API-Football** (보충, Ultra $29/mo) — League ID: 39, SportMonks에서 커버되지 않는 H2H 상세, 예측, 추가 스탯 보충

### 기타

- **date-fns** — 날짜 포매팅
- **sonner** — 토스트 알림
- **next-themes** — 다크/라이트 모드

### 배포

- **Vercel** — Next.js 16+ 최적화, Vercel Cron (일간/주간 데이터 동기화)
