# PL 데이터 플랫폼 MVP 기획서

## Context

프리미어리그(PL) 팬들은 선수/경기 데이터를 확인하기 위해 평균 3~5개 사이트를 방문해야 한다. FBref, WhoScored, Transfermarkt, SofaScore 등 데이터가 파편화되어 있고, 고급 지표(xG, xA 등)는 맥락 없이 숫자만 나열되어 대다수 팬이 해석하기 어렵다. **"흩어진 데이터를 한 곳에, 숫자를 이야기로"** 바꾸는 프로젝트를 기획한다.

---

## 1. 프로젝트 정의

| 항목           | 내용                                                          |
| -------------- | ------------------------------------------------------------- |
| **프로젝트명** | (추후 결정)                                                   |
| **한 줄 설명** | PL 선수·경기 데이터를 맥락과 함께 쉽고 재밌게 보여주는 플랫폼 |
| **핵심 가치**  | 파편화 해소 · 맥락 해석 · 시각적 재미 · 공유 가능성           |
| **타겟 유저**  | 모든 PL 팬 (캐주얼 ~ 열성 팬 모두 포함)                       |
| **MVP 범위**   | 선수 프로필 + 선수 비교 배틀카드 + 매치데이 대시보드          |

---

## 2. 핵심 문제 정의

### 문제 1: 데이터 파편화 (10개 탭 문제)

- 선수 정보를 완전히 파악하려면 최소 3~5개 사이트 필요
- 각 사이트마다 다른 UI, 다른 지표, 다른 기준
- **목표**: 한 화면에서 선수의 모든 핵심 데이터 확인

### 문제 2: 맥락 없는 숫자 (숫자의 벽)

- "xG 14.2"가 좋은 건지 나쁜 건지 대부분의 팬이 모름
- 리그 평균, 포지션 평균, 전년 대비 등 비교 기준 부재
- **목표**: 모든 숫자에 "그래서 이게 어느 수준인데?"를 답해주기

### 문제 3: 비교의 어려움

- 팬 논쟁의 핵심인 선수 비교를 데이터로 하기 어려움
- 기존 비교 도구는 스프레드시트 수준의 UX
- **목표**: 시각적이고 공유 가능한 비교 경험

### 문제 4: 경기 전/중/후 맥락 부족

- 경기 보기 전 "뭘 알고 보면 더 재밌는지" 정보가 분산
- **경기 중 "지금 무슨 상황인지" 데이터로 파악하기 어려움**
  - 중계 화면만으로는 경기 흐름(모멘텀)이 어느 쪽인지 감으로만 판단
  - 특정 선수가 얼마나 뛰고 있는지, 어디서 활동하는지 보이지 않음
  - 전술 변화(포메이션 변경, 선수 역할 변화)를 실시간으로 인지하기 어려움
- 경기 후 핵심 데이터 요약이 없음
- **목표**: 경기 전→중→후 풀사이클 데이터 기반 인사이트 제공

## 3. 핵심 기능 상세

### 기능 A: 선수 프로필 원스톱

**컨셉**: 선수 이름 하나로 "이 선수의 모든 것"을 한 화면에

#### 화면 구성

**상단 - 선수 카드**

- 선수 사진, 이름, 팀, 포지션, 등번호, 국적
- 시즌 핵심 요약 한 줄: "올 시즌 리그 득점 3위, 작년 대비 폼 상승 중"

**섹션 1 - 시즌 핵심 스탯 (컨텍스트 카드)**

```
┌─────────────────────────────────────────┐
│  ⚽ 골: 15       📊 리그 3위 / 상위 2%  │
│  🎯 xG: 12.8     → 실제 골 > xG (결정력↑)│
│  🅰️ 어시스트: 8   📊 리그 7위 / 상위 5%  │
│  📈 작년 동기 대비: 골 +4, 어시스트 +2    │
└─────────────────────────────────────────┘
```

- **핵심 차별점**: 모든 숫자 옆에 리그 내 순위, 백분위, 전년 비교를 함께 표시
- 지표 클릭 시 "이 지표가 뭔가요?" 팝오버 설명 제공

**섹션 2 - 능력치 레이더 차트**

- 공격, 창의성, 수비 기여, 패스, 드리블, 체력 등 6축 레이더
- 같은 포지션 리그 평균과 오버레이 비교
- "이 선수의 강점은 OOO, 약점은 OOO" 자동 요약

**섹션 3 - 폼 트렌드 (최근 10경기)**

- 라인 차트: 경기별 평점/기여도 변화 추이
- "최근 5경기 폼: 🔥 상승세" 같은 직관적 라벨

**섹션 4 - 포지션 내 랭킹**

- 같은 포지션 선수 중 주요 지표별 순위
- "PL 윙어 중 드리블 성공률 1위, 슈팅 정확도 4위"

---

### 기능 C: 선수 비교 배틀카드

**컨셉**: 선수 2~3명을 선택하면 시각적 비교 카드를 자동 생성

#### 화면 구성

**선수 선택 UI**

- 직관적인 선수 검색/선택 (최대 3명)
- 인기 비교 조합 추천: "이번 주 인기 비교: 살라 vs 손흥민 vs 사카"

**비교 카드 레이아웃**

```
┌──────────────────────────────────────────┐
│         살라  vs  손흥민  vs  사카         │
│                                          │
│  [레이더 차트 - 3명 오버레이]              │
│                                          │
│  ⚽ 골     │ 15 🏆 │  11  │  13  │       │
│  🎯 xG    │ 12.8  │  9.2 │ 11.5 │       │
│  🅰️ 어시  │  8 🏆 │   7  │   6  │       │
│  📊 평점   │ 7.4   │ 7.2  │ 7.6 🏆│      │
│  🏃 드리블 │  52   │ 38   │ 67 🏆 │      │
│                                          │
│  💬 AI 한줄평:                            │
│  "살라는 득점력, 사카는 돌파력, 손흥민은     │
│   공수 밸런스에서 우위"                     │
│                                          │
│  [📤 이미지로 공유] [🔗 링크 복사]          │
└──────────────────────────────────────────┘
```

**핵심 차별점**:

- 🏆 아이콘으로 각 지표별 "승자" 즉시 식별
- AI 기반 한줄평으로 숫자를 해석
- **SNS 공유용 이미지 자동 생성** → 팬 커뮤니티 바이럴 유도

### 기능 B: 매치데이 대시보드

**컨셉**: "경기 전에 예습하고, 경기 중에 이해하고, 경기 후에 복기하는" 데이터 동반자

#### 화면 구성 - 경기 전 (프리매치)

**라운드 전체 뷰**

```
┌─────────────── GW 28 ─────────────────┐
│                                        │
│  🔥 이번 라운드 주목 경기               │
│  ┌────────────────────────────────┐    │
│  │ 아스널 vs 맨시티               │    │
│  │ 우승 경쟁 빅매치 | H2H: 아스널 우세│  │
│  │ 핵심 매치업: 사카 vs 워커       │    │
│  │ [자세히 보기]                   │    │
│  └────────────────────────────────┘    │
│                                        │
│  📋 전체 경기 목록                      │
│  리버풀 vs 첼시   | 순위 영향도: ★★★★  │
│  토트넘 vs 에버턴 | 손흥민 폼: 🔥       │
│  뉴캐슬 vs 울브스 | 이사크 득점왕 레이스 │
│  ...                                   │
└────────────────────────────────────────┘
```

**개별 경기 프리매치 카드**

- 양팀 최근 5경기 폼 (W/D/L 시각화)
- 상대 전적 (최근 5회)
- 핵심 대결 구도 (선수 매치업)
- 순위 시뮬레이션: "이기면 O위, 지면 O위"
- 부상/결장 선수 정보

#### 화면 구성 - 경기 중 (라이브 인사이트)

**컨셉**: 중계 화면 옆에 두고 보는 "경기 해설 대시보드"

- 팬이 TV/스트리밍으로 경기를 보면서, 폰이나 세컨드 스크린으로 실시간 데이터를 확인
- 중계에서 보여주지 않는 이면의 데이터를 시각적으로 제공

**모멘텀 그래프 (핵심 기능)**

```
┌──────────────────────────────────────────┐
│  ⚡ 경기 모멘텀                           │
│                                          │
│  아스널 ████████████▓░░░░░░░░ 맨시티      │
│                                          │
│      [모멘텀 타임라인 그래프]              │
│  아스널 ──╲    ╱──╲         ╱──          │
│            ╲╱    ╲───╲   ╱              │
│  맨시티              ╲─╱                 │
│  ──────────────────────────────────────  │
│  0'   15'   30'   45'  60'  75'  90'    │
│                                          │
│  📍 현재: 맨시티 우세 (최근 10분 슈팅 4회) │
│  ⚽ 58' 살라 골! → 모멘텀 급변            │

```

- 시간대별 공격 흐름을 시각화 (슈팅, 점유율, 진영 비율 기반)
- 골, 카드, 교체 등 이벤트 발생 시 모멘텀 그래프 위에 마커 표시
- "현재 어느 팀이 밀고 있는지" 한눈에 파악 가능

- 모든 주요 이벤트에 **데이터 맥락** 추가 (골의 xG, 교체의 전술적 의미)
- 전술 변화 감지: 포메이션 변경 시 자동 알림

#### 화면 구성 - 경기 후 (포스트매치)

**경기 결과 + 데이터 리뷰**

- 스코어 + 핵심 스탯 비교 (점유율, xG, 슈팅 등)
- MOM(Man of the Match) 데이터 근거 선정
- "이 경기 핵심 순간" 타임라인 (골, 레드카드, 전술 변화)
- "이 결과로 순위는 이렇게 바뀌었다" 시각화

## 4. UX 설계 원칙

### 원칙 1: "모든 숫자에 맥락을"

- 단독 숫자 표시 금지. 반드시 비교 대상과 함께 표시
- 예: "15골" → "15골 (리그 3위, 상위 2%, 작년 대비 +4)"

### 원칙 2: "3초 안에 핵심 파악"

- 화면 진입 후 3초 안에 핵심 정보를 파악할 수 있어야 함
- 색상, 아이콘, 크기로 정보의 우선순위를 시각화
- 상세 데이터는 깊이 들어갈수록 점진적으로 공개 (Progressive Disclosure)

### 원칙 3: "공유하고 싶은 디자인"

- 비교 카드, 선수 프로필 요약 등은 SNS 공유 최적화
- 카드 이미지 자동 생성 (OG 이미지, 다운로드)
- "이 데이터 공유하기" 버튼을 핵심 위치에 배치

### 원칙 4: "초보 팬도 환영"

- 전문 용어에는 항상 설명 팝오버 제공
- "xG란? 선수의 슈팅 위치/상황을 고려한 기대 득점 수치"
- 복잡한 차트보다 직관적 비유 사용: "리그 상위 5% → 20명 중 1등급"

## 5. 핵심 데이터 포인트

### 선수 데이터

| 카테고리 | 지표                           | 맥락화 방식              |
| -------- | ------------------------------ | ------------------------ |

## 7. 사용자 시나리오

### 시나리오 1: 캐주얼 팬 "민수" (25세)

> "토요일 밤 EPL 보기 전에 이번 라운드 뭐가 재밌는지 알고 싶어"

1. 매치데이 대시보드 접속
2. "이번 라운드 주목 경기" 확인 → 아스널 vs 맨시티
3. 프리매치 카드로 양팀 폼, 핵심 매치업 파악
4. "사카 vs 워커" 매치업이 재밌어보여서 사카 프로필 클릭
5. 사카의 이번 시즌 성적과 폼 추이 확인
6. **경기 시작 → 라이브 인사이트 화면으로 자동 전환**
7. **TV로 경기 보면서 폰으로 모멘텀 그래프 확인 → "아 지금 맨시티가 밀고 있구나"**
8. **교체 발생 → 타임라인에서 전술 변화 의미 확인**
9. 경기 종료 후 → 포스트매치 데이터 리뷰 확인

### 시나리오 2: 열성 팬 "지연" (30세)

> "커뮤니티에서 살라 vs 손흥민 논쟁 중인데 근거가 필요해"

1. 선수 비교 → 살라, 손흥민 선택
2. 레이더 차트로 한눈에 강약점 비교
3. 각 지표별 상세 비교 확인
4. AI 한줄평 확인
5. **배틀카드 이미지 저장 → 커뮤니티에 공유**

### 시나리오 3: 입문 팬 "현우" (22세)

> "xG가 뭔지 모르겠는데, 선수 성적을 좀 더 깊이 알고 싶어"

1. 좋아하는 선수 검색 → 손흥민 프로필
2. 기본 스탯 (골, 어시스트)은 바로 이해
3. xG 옆 "?" 클릭 → "기대골이란 이러저런 개념입니다" 설명
4. "리그 상위 15%" 같은 맥락 정보로 숫자의 의미 파악
5. 점점 더 많은 지표를 이해하게 됨 → 데이터 리터러시 성장

## 8. MVP 기능 우선순위

### P0 (반드시 포함)

- [ ] 선수 검색 및 프로필 페이지 (핵심 스탯 + 맥락)
- [ ] 선수 비교 (2명) + 배틀카드 이미지 생성
- [ ] 매치데이 대시보드 - 프리매치 (라운드별 경기 목록 + 프리매치 카드)
- [ ] 매치데이 대시보드 - 라이브 (모멘텀 그래프 + 라이브 스탯 비교 + 이벤트 타임라인)
- [ ] 지표 설명 팝오버

### P1 (MVP 이후 빠르게 추가)

- [ ] 라이브 히트맵 (팀/개별 선수)
- [ ] 실시간 선수 퍼포먼스 패널 (평점 + 핵심 액션)
- [ ] 포스트매치 데이터 리뷰
- [ ] 선수 비교 3명 확장
- [ ] 선수 폼 트렌드 차트
- [ ] SNS 공유 최적화 (OG 이미지)

### P2 (향후 확장)

- [ ] xG 흐름 차트 (시간대별 누적 xG)
- [ ] 포메이션 변화 자동 감지 및 알림
- [ ] 시즌 스토리라인 (득점왕 레이스, 강등 경쟁 등)
- [ ] 커뮤니티 기능 (인기 비교 랭킹)
- [ ] 알림 기능 (관심 선수 폼 변화)
- [ ] AI 기반 경기 예측

## 9. 검증 방법

### 핵심 가설

1. PL 팬은 파편화된 데이터를 한 곳에서 보고 싶어한다
2. 맥락이 있는 숫자가 맥락 없는 숫자보다 만족도가 높다
3. 비교 배틀카드는 SNS 공유를 통해 바이럴된다

### MVP 성공 지표

- 선수 프로필 페이지 평균 체류 시간 > 2분
- 비교 카드 생성 후 공유 전환율 > 15%
- 매치데이 대시보드 주간 재방문율 > 40%
- **라이브 인사이트 경기 중 동시접속 유지율 > 60% (경기 시작~종료)**
- **라이브 인사이트 세션당 평균 체류 시간 > 경기 시간의 50%**
